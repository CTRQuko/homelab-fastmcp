# Código de conducta

🇬🇧 [Read in English](CODE_OF_CONDUCT.md) · 🇪🇸 Estás leyendo la versión en español

Este proyecto adopta el **Contributor Covenant, versión 2.1**, sin
modificaciones. El texto completo está publicado en:

<https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/>

Un resumen breve en nuestras palabras:

- Trata a otras contributors como colegas. Discrepar está bien;
  el desprecio no.
- Las revisiones se centran en el código, el diseño, el issue —
  no en la persona.
- Las contribuciones nuevas son bienvenidas. *"¿Por qué no leíste
  los docs?"* no es una respuesta.
- El maintainer tiene la última palabra sobre el scope. El
  push-back está bien, las pile-ons no.

## Reportar

Para temas de Código de Conducta, contacta al maintainer en
privado. La función "private vulnerability reporting" de GitHub
funciona como canal privado incluso para reportes que no son de
seguridad — abre el repo, clica *Security* → *Private vulnerability
reporting*.

El maintainer responde dentro de una semana. La confidencialidad
se respeta a menos que el reporter diga lo contrario.

## Aplicación

Aplica el escalado estándar del Contributor Covenant (corrección
→ aviso → ban temporal → ban permanente). El texto exacto vive en
el documento upstream enlazado arriba.
